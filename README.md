# RingkasanBuku

Aplikasi ringkasan buku berbasis Java + JavaFX.

## Requirements
- Java 17+
- JavaFX SDK 26.0.1 → download di https://gluonhq.com/products/javafx/
- iTextPDF 5.5.13.4 → download di https://mvnrepository.com/artifact/com.itextpdf/itextpdf/5.5.13.4

## Setup
1. Clone repo ini
2. Download JavaFX SDK dan letakkan di folder `lib/javafx-sdk-26.0.1/`
3. Download iTextPDF jar dan letakkan di `lib/itextpdf-5.5.13.4.jar`
